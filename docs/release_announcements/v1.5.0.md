# flux v1.5.0 is now live

We’ve completed the `v1.5.0` ecosystem release for **flux**.

`flux` is a modular R ecosystem for simulation modeling of probabilistic dynamic systems in irregular time, with coordinated packages for:

- core simulation engine
- train/test/validation (TTV) data preparation
- forecasting and summary estimators
- prediction-vs-observed validation
- multi-process orchestration
- reference model and template scaffolding

## Highlights in v1.5.0

- ecosystem harmonization under `flux*` package/repo naming
- public API naming cleanup and documentation alignment
- expanded manual `.Rd` coverage and signature consistency
- cross-package ecosystem test harness with tiered checks
- consolidated one-file-per-run ecosystem reporting
- top-level super-repo with coordinated release tooling
