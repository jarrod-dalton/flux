# Work In Progress

This folder contains exploratory material that is not yet part of the stable
user-facing tutorials or package-shipped interfaces.

Current contents:

- `book/`: planned long-form bookdown content and scaffold files.
- `api/`: API prototypes and drafts for future formalization.

Guideline:

- If content is intended for stable onboarding/teaching, promote it to
  `tutorials/`.
- If content is intended to ship as package behavior, move it into the relevant
  package only after contracts, tests, and documentation are in place.
